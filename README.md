# Nuntium News App

### News App built with React Native / Typescript.

## Features

- Signup
  - Select Favourite Topic
- Signin
- Forgot Password
  - Enter Email
  - Enter Verification Code
- Browse News
- Categories
- Profile
  - Change Password
  - Language
  - Terms
  - Privacy
- Bookmarks
